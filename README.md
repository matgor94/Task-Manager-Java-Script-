# Task Manager App

## PL - Opis aplikacji

Aplikacja **Task Manager** to prosty menedżer zadań stworzony w JavaScript, działający z lokalnym serwerem JSON (JSON Server).  
Pozwala użytkownikowi na:

- Logowanie i rozpoznawanie zalogowanego użytkownika (po emailu),
- Dodawanie nowych zadań przypisanych do użytkownika,
- Usuwanie zadań,
- Oznaczanie zadań jako „Finished”,
- Wyświetlanie tylko zadań należących do aktualnie zalogowanego użytkownika,
- Dynamiczną numerację zadań w tabeli.

Aplikacja została zaprojektowana tak, aby nauka pracy z **fetch API** i asynchronicznością była łatwa i przejrzysta.

---

## EN - Application Description

The **Task Manager** app is a simple task management tool built with JavaScript, working with a local JSON server (JSON Server).  
It allows the user to:

- Log in and identify the logged-in user (by email),
- Add new tasks assigned to the user,
- Delete tasks,
- Mark tasks as "Finished",
- Display only tasks belonging to the currently logged-in user,
- Dynamically number tasks in the table.

The application is designed to make learning **fetch API** and asynchronous operations easy and clear.
